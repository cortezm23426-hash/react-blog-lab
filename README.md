# React Blog Site

This project is a simple personal blog built using React. The purpose of this lab is to practice building React components and passing data using props.

## Component Structure

App
├ Header
├ About
└ ArticleList
    └ Article

## Features

- Displays a blog title
- Shows a blog image and description
- Renders multiple articles dynamically
- Uses props to pass data between components

## Technologies Used

- React
- JavaScript
- Vite

## How to Run the Project

1. Install dependencies

npm install

2. Start the development server

npm run dev

3. Open the browser at the URL shown in the terminal (usually http://localhost:5173)
